hello skids
this is a very basic starter base
it includes custom memory in usermode a simple render(keybind f1) system and basic visuals (box only)
theres also a lightweight cache system built on top of memory reads nothing here is polished its just a starter foundation, not a finished project
if you want to improve or understand parts of it check the win32 tlhelp32 api: https://learn.microsoft.com/en-us/windows/win32/api/tlhelp32/
use it if youre working on process snapshots module walking or anything related to basic memory enumeration dont expect anything advanced here its meant to be a learning base
